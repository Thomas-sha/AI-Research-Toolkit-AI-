## 这是什么 | What This Is

一套面向市场研究从业者的AI增强方法论Skill集合。每个Skill包含：

- 方法论核心思想与标准流程
- AI加速的具体环节与工具链
- 人工判断的不可替代节点
- 脱敏的真实案例参考
- 可直接复用的输出模板

A collection of AI-augmented methodology skills for market research professionals. Each skill contains core methodology, AI acceleration points, irreplaceable human judgment nodes, anonymized case references, and reusable templates.

## 使用方式 | How to Use

### Claude Code 用户

将 Skill 文件放入你的 Claude Code skills 目录：

```
cp skills/*.md ~/.claude/skills/
```

触发方式：描述你的研究需求，Claude 会自动加载对应 Skill。

### 通用用户

直接阅读 Skill 文件中的方法论和模板，手动应用到你的研究流程中。
